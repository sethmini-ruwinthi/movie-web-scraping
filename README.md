# Movie Web Scraping

## Description
This project demonstrates web scraping using Python to collect movie-related data from a public website.

## Features
- Sending HTTP requests
- Parsing HTML content
- Extracting movie details
- Saving scraped data into a structured format

## Tools & Libraries
- Python
- Requests
- BeautifulSoup
- Pandas

## Purpose
This project is created for learning and portfolio purposes.
